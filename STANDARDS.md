# 01 — Project Overview

This project is a professional landing page for Sneha Saravanan, a Business Analytics and Information Systems student at the University of Iowa.

The site serves as a dynamic portfolio to demonstrate technical proficiency in Python, SQL, and Power BI to recruiters in finance, consulting, and operations.

The goal is to provide context for high-impact achievements, such as the 15–20% efficiency gains realized during the DHL Supply Chain internship.

---

# 03 — Design Standards

* Clean, professional, recruiter-friendly layout
* Avoid clutter and excessive animations
* Ensure fast scannability (key info visible in under 10 seconds)
* Maintain consistent spacing and alignment

---

# 04 — Typography

**Font Stack:**

* Inter, system-ui, Arial

**Hierarchy:**

* H1: 2.5rem – 3rem (Bold)
* H2: 1.8rem – 2rem (Semi-bold)
* Body: 1rem (Line-height: 1.6)

**Color:**

* Use #333333 instead of pure black

---

# 05 — Color Palette

* Primary: #003366 (Navy Blue)
* Accent: #FFCD00 (Gold/Yellow)
* Background: #FFFFFF (White)
* Surface: #F8F9FA (Light gray sections)

---

# 06 — Component Guidelines

**Navbar:**

* Fixed top navigation
* Links: About, Skills, Projects, Contact

**Sections:**

* Max width: 1100px
* Centered layout

**Project Cards:**

* Title
* Tools used
* Key result (bullet point)
* Subtle border or shadow

---

# 07 — Responsiveness

* Mobile-first design
* Use Flexbox for layout
* Stack elements on small screens
* Breakpoint at 768px

---

# 08 — Code Standards

* Use semantic HTML (`<header>`, `<section>`, `<footer>`)
* Add comments in CSS
* Use clear class names (e.g., `.project-card`)
* Define variables in `:root`
